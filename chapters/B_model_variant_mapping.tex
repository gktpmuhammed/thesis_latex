\section{Legacy Aliases and Implementation-Only Variant}
\begin{itemize}
    \item \path{medical_vlm_8_tokens_full} is a deprecated duplicate branch of the current Base-8T setup and is excluded from active comparison tables.
    \item \path{lora_with_vis_tokens_pos_embed} is an older naming alias of the Base-1T branch in legacy result exports.
    \item \path{medgemma_lora_vis_token_pos_embed} is an older naming alias of the Base-1T branch in legacy result exports.
    \item \path{medgemma_lora_vis_token} corresponds to a single-token interface prototype (Base-1T) kept for implementation comparison; it is not included in the standardized training metric tables.
\end{itemize}
