\section{Legacy Aliases and Implementation-Only Variant}
\begin{itemize}
    \item \path{medgemma_vlm_8_tokens_full} is an older naming alias of the Baseline-8T branch in legacy result exports.
    \item \path{lora_with_vis_tokens_pos_embed} is an older naming alias of the Prefix-Stable-1T branch in legacy result exports.
    \item \path{medgemma_lora_vis_token} corresponds to a single-token interface prototype (Baseline-1T) kept for implementation comparison; it is not included in the standardized retraining metric tables.
\end{itemize}
